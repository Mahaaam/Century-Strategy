# نقشه قرن و استراتژی قدرت

سایت تحلیلی پرونده استراتژیک آمریکا و ایران

## ساختار پروژه

- `index.html` - صفحه اصلی با ناوبری
- `styles.css` - طراحی تم دارک و ریسپانسیو
- `script.js` - مدیریت ناوبری و بارگذاری محتوا
- `Mobile.png` - تصویر صفحه اصلی برای موبایل
- `desktop.png` - تصویر صفحه اصلی برای دسکتاپ
- فایل‌های `.md` - محتوای ۶ مرحله تحلیل

## نحوه اجرا

### روش ۱: باز کردن مستقیم
فایل `index.html` را در مرورگر باز کنید.

### روش ۲: با سرور محلی
```bash
npx http-server -p 8000
```
سپس به آدرس `http://localhost:8000` بروید.

## ویژگی‌ها

- صفحه لندینگ ریسپانسیو با تصاویر متناسب با دستگاه
- فهرست ۶ مرحله تحلیلی
- صفحات نمایش محتوا با طراحی خوانا
- تم دارک حرفه‌ای
- فونت فارسی Vazirmatn
- انیمیشن‌های ظریف
- کاملاً ریسپانسیو برای موبایل و دسکتاپ
